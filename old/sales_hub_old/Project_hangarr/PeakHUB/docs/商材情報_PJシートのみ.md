# ピークハブ 商材情報（PJシート要約・統合）

## 1. 商材サマリー

- **商材名**: RPO（採用代行／採用コンサル）
- **提供価値（Who/What）**: 人事機能はあるが採用ノウハウ不足で実務が回っていない企業に対し、採用の**戦略設計から運用まで**を一気通貫で支援する

## 2. 提供範囲（できること）

- **戦略設計**
- **運用代行**
  - 日程調整
  - 面接の代行

## 3. 想定ターゲット（誰に売るか）

- **企業規模**: 中小企業・スタートアップ
- **刺さりやすい顧客像**: 人事はいるが採用ノウハウ不足で、採用業務が回っていない企業

## 4. 価格感（資料内の例）

- **提供形態**: 人月での提供（小さめの案件）
- **価格例**: 3か月 90万円（= 月30万円）
- **成果報酬**: 原則不可（条件次第で可能性はあるが、現状は無し）

## 5. 競合との差別化・勝ちポイント（メモ統合）

- **主訴求の前提**: 「実務レベルのノウハウがない」層に対する支援
- **チャネル/媒体メモ**
  - **Indeed**: 小さい金額でも運用しやすく、コスパよく母集団形成しやすい
  - **ビズリーチ（ハイクラス）**: ターゲット設計・メッセージングで返信率を高めやすい
- **差別化アイデア**: お試しのようなフロント商材

## 6. 既存でうまくいっているパターン（メモ）

- 人事はいるがノウハウがない会社への支援

## 7. 狙い先セグメント仮説（メモ統合）

- **大手採用メディア掲載企業の上位掲載企業**: 採用予算があり、採用ニーズが高い
- **IT・SaaS企業**: 採用ニーズがあり、採用投資意識が高く、予算も比較的潤沢
- **医療・ヘルスケア（病院グループ、調剤薬局チェーン、医療機器メーカー等）**: 慢性的な人手不足で専門職アプローチが必要（支援実績メモあり）
- **ホテル業界**: バイネームリスト運用のメモあり
- **製造業（準大手・中堅）**: 技術者・エンジニア確保、母集団形成に課題（アポ数が落ちる可能性メモあり）
- **物流・運輸**: ドライバー不足が深刻、業界課題が深い
