# 報告テンプレート（PJシート抜粋）

出典: `PeakHub_PJシート.xlsx`（PJシート）内の記載から、フロー上で使う「報告・連絡系」のテンプレを抜粋して整理

## 1. 稼働開始報告（Slack）

```
＝＝＝＝＝
@マネージャー @ディレクター
おはようございます！
本日の作業開始いたします。
9:00-10:00 13:00-18:00
＝＝＝＝＝
```

補足: 稼働終了後は日報を19時までに報告

## 2. 日報（業務終了時）

```
お世話になっております。
本日の日報をお送りいたします。

稼働日　　　　  ：X/X(X)
稼働時間　　　  ：XXh
　┗架電時間　  ：
架電件数　　　  ：X件
受付NG数　　　：X件
キーマン接続数  ：X件
キーマンNG数　：X件
アポイント数　  ：X件
資料送付数　　  ：X件

▼架電以外の業務内容(かかった時間も記載)

▼受付NG理由、キーマンNG理由記載

▼上手くいった施策

▼次回架電への施策
```

運用ルール（PJシート記載）:

- 稼働日19時までに報告
- 業務終了時に必ず提出
  - 日報提出
  - 実績入力（KPI設計タブ）
